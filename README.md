# 🌌 Qr Studio 
**The ultimate high-performance QR & Barcode generation suite.**

Anik Studio | Forge is a minimalist, glassmorphic digital tool designed for professionals. It allows users to generate high-fidelity QR codes with custom branding and industrial-standard barcodes directly from their mobile devices.

---

## 💎 Features
* **QR Forge:** Generate high-resolution QR codes with "Liquid" or "Sharp" node styles.
* **Custom Branding:** Seamlessly upload and embed your brand logo into the center of the QR code.
* **Industrial Barcodes:** Support for CODE128 and EAN-13 protocols.
* **Kinetic UI:** A dynamic, deep-space animated background with a glassmorphic interface.
* **Mobile-First Design:** Fully optimized for iOS and Android "Safe-Zones" to prevent navigation interference.

---

## 🛠️ Tech Stack
This project is built with raw performance and zero-dependency bloat in mind:
* **Logic:** JavaScript (ES6+)
* **Styling:** CSS3 (Custom Variables & Backdrop Filters)
* **Structure:** HTML5 (Semantic & Mobile-Responsive)
* **Libraries:** * `qr-code-styling` (Premium QR rendering)
    * `JsBarcode` (Industrial barcode logic)
    * `FontAwesome` (Vector iconography)

## 🚀 Deployment Instructions

To host this on your own GitHub Pages:

1.  **Fork** this repository.
2.  Ensure your file structure looks like this:
    ```text
    ├── index.html
    ├── style.css
    ├── script.js
    └── README.md
    ```
3.  Go to **Settings > Pages**.
4.  Select **Branch: main** and folder **/(root)**.
5.  Click **Save**. Your studio will be live at `https://yourusername.github.io/your-repo-name/`.
